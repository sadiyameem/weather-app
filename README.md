# weather-app

# Description
- This is a simple web-based app where you can see the weather in any city.

# Features
- Temperature
- Humidity
- Wind speed
- Button
- Text input box
- Card where all the weather information is present

# Tools
- HTML
- CSS
- JavaScript
- openweathermap for the API

# Challenges/Learned
- I learned how to use an API to generate weather info
- A challenge was to find the information I needed in the API website
